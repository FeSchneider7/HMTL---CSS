Principais formatações

Negrito ou Destaque

Nessa frase temos um termo em NEGRITO usando a tag B (não semantica)

Nesta frase tem um TERMO EM DESTAQUE usando a TAG Strong que é semantica

Ítalico ou Enfase

Nesta frase temos um termo em ítalico usando a TAG I

ou usanto CTRL + SHIFT + P para incluir o Italico seleciondo o texto

Nesta frase temos enfase usando a TAG EM que é uma TAG semantica

Texto Marcado

podemos criar um texto MARCADO, usando a tag MARK

E em outro paragrafo temos outro texto marcado aqui no final.

Texto deletado

Tag DEL - Podemos marcar um texto como EXCLUÍDO para indicar que ele deve ser lido, mas nao considerado

Texto inserido

Tag INS - Podemos marcar um texto para dar enfase e indicar que ele foi ADCIONADO depois

Temos também a TAG

Texto Sobrescrito

Tag SUP - Para inserir coisas tipo: x20+3 Para ficar assim x20+3

Texto Subscrito

Tag SUB - Para inserir coisas do tipo H2O para ficar assim H2O
