Trabalhando com Listas

Listas Ordenadas

Para fazer uma lista ordenada basta usar a Tag OL e depois ir listando cada item com a Tag LI, isso ajuda pois se precisar incluir ou excluir um item da lista o proprio sistema altera a ordem para voce, conforme abaixo:

  1. Acordar
  2. Tomar cafe
  3. lavar o prato
  4. Ir para academia
  5. voltar
  6. tomar banho

Listas não Ordenadas
